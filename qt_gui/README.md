# Qt GUI для системы алгебры полиномов

Графический интерфейс пользователя для работы с полиномами, построенный на Qt6.

## Требования

- Qt6 (Core, Widgets)
- CMake 3.16+
- C++17 совместимый компилятор

## Установка Qt6

### macOS (Homebrew)
```bash
brew install qt6
```

### Ubuntu/Debian
```bash
sudo apt-get install qt6-base-dev qt6-tools-dev
```

### Windows
Скачайте Qt6 с официального сайта: https://www.qt.io/download

## Сборка

1. Убедитесь, что Qt6 установлен и доступен в PATH
2. Из корневой директории проекта:

```bash
mkdir build
cd build
cmake .. -DBUILD_QT_GUI=ON
make
```

Если Qt6 не найден автоматически, укажите путь:
```bash
cmake .. -DBUILD_QT_GUI=ON -DQt6_DIR=/path/to/qt6/lib/cmake/Qt6
```

## Запуск

```bash
./qt_gui/polynomial_qt_gui
```

## Возможности

### Вкладка "Полиномы"
- Создание и редактирование полиномов
- Добавление мономов с дробными степенями
- Дифференцирование и интегрирование
- Сохранение и загрузка полиномов

### Вкладка "Таблицы"
- Переключение между 6 типами таблиц:
  - Linear Array Table
  - Linear List Table
  - Ordered Array Table
  - AVL Tree Table
  - Hash Table (Open Addressing)
  - Hash Table (Chaining)
- Просмотр содержимого таблиц
- Поиск полиномов
- Статистика таблиц

### Вкладка "Выражения"
- Вычисление арифметических выражений
- Поддержка операций: +, -, *, ( )
- Использование имен полиномов в выражениях
- Сохранение результатов
- История вычислений

## Примеры использования

1. **Создание полинома:**
   - Перейдите на вкладку "Полиномы"
   - Введите имя полинома
   - Добавьте мономы, указав коэффициент и степени x, y, z
   - Нажмите "Сохранить полином"

2. **Вычисление выражения:**
   - Перейдите на вкладку "Выражения"
   - Введите выражение, например: `pol1 + pol2 * 2.5`
   - Нажмите "Вычислить" или "Вычислить и сохранить"

3. **Работа с таблицами:**
   - Перейдите на вкладку "Таблицы"
   - Выберите тип таблицы из выпадающего списка
   - Просматривайте содержимое и статистику

## Архитектура

- **MainWindow** - главное окно приложения
- **PolynomialWidget** - виджет для работы с полиномами
- **TableWidget** - виджет для управления таблицами
- **ExpressionWidget** - виджет для вычисления выражений

Все виджеты интегрированы с существующей системой полиномов через `TPolinomManager`.

## Темная тема

Приложение использует темную тему по умолчанию для лучшего восприятия математических выражений.

## Горячие клавиши

- **Ctrl+N** - Новый полином
- **Ctrl+O** - Открыть файл
- **Ctrl+S** - Сохранить
- **Ctrl+Q** - Выход
- **Enter** - Вычислить выражение (в поле ввода)

## Устранение неполадок

### Qt6 не найден
```bash
export Qt6_DIR=/path/to/qt6/lib/cmake/Qt6
```

### Ошибки компиляции MOC
Убедитесь, что CMake правильно настроен для Qt6:
```cmake
set(CMAKE_AUTOMOC ON)
set(CMAKE_AUTOUIC ON)
set(CMAKE_AUTORCC ON)
```

### Проблемы с темной темой
Если темная тема не применяется, проверьте поддержку стиля Fusion в вашей системе.
