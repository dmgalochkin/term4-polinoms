# Polynomial Library Documentation

## Обзор

Библиотека для работы с многочленами от трех переменных (x, y, z) с дробными показателями степеней. Включает в себя классы для мономов, полиномов, таблиц для хранения полиномов и графический интерфейс на Qt6.

## Основные классы

### TMonom
Класс для представления одночлена (монома) от трех переменных с дробными показателями степеней.

**Основные возможности:**
- Создание мономов с коэффициентом и степенями переменных
- Арифметические операции (умножение, умножение на скаляр)
- Дифференцирование и интегрирование по любой переменной
- Сравнение мономов для сортировки и приведения подобных

**Пример использования:**
```cpp
// Создание монома 2.5 * x^0.5 * y^1.3 * z^0
TMonom m1(2.5, {0.5, 1.3, 0.0});

// Дифференцирование по x (переменная 0)
TMonom dm = m1.differentiate(0);

// Интегрирование по y (переменная 1)
TMonom im = m1.integrate(1);
```

### TPolinom
Класс для представления многочлена как суммы мономов.

**Основные возможности:**
- Создание полиномов из мономов или списков мономов
- Арифметические операции (+, -, *, умножение на скаляр)
- Дифференцирование и интегрирование
- Вычисление значения в точке
- Автоматическое приведение подобных и нормализация

**Пример использования:**
```cpp
// Создание полинома P = 2.5*x^0.5*y^1.3 - 3.0*x^2
TMonom m1(2.5, {0.5, 1.3, 0.0});
TMonom m2(-3.0, {2.0, 0.0, 0.0});
TPolinom P({m1, m2});

// Дифференцирование по x
TPolinom dP = P.differentiate(0);

// Вычисление в точке (1.0, 2.0, 3.0)
double value = P.evaluate({1.0, 2.0, 3.0});
```

### TTable<T>
Абстрактный интерфейс для таблиц, хранящих элементы типа T.

**Основные методы:**
- `insert(key, value)` - вставка элемента
- `find(key)` - поиск элемента
- `remove(key)` - удаление элемента
- `clear()` - очистка таблицы
- `size()` - размер таблицы

### TArrayTable<T>
Реализация таблицы на основе динамического массива.

**Особенности:**
- Линейный поиск O(n)
- Динамическое изменение размера
- Простая реализация для небольших объемов данных

### TPostfixCalculator
Калькулятор для вычисления постфиксных выражений с полиномами.

**Поддерживаемые операции:**
- Арифметические: +, -, *, /
- Функции: diff(var), int(var)
- Переменные полиномов из таблицы

### TPolinomManager
Центральный менеджер для управления полиномами и таблицами.

**Функциональность:**
- Управление таблицами полиномов
- Вычисление выражений
- Координация между компонентами системы

## Графический интерфейс

### MainWindow
Главное окно приложения с тремя вкладками:
- **Polynomials** - создание и редактирование полиномов
- **Tables** - управление таблицами полиномов
- **Expressions** - вычисление выражений

### PolynomialWidget
Виджет для работы с полиномами:
- Создание полиномов из мономов
- Редактирование коэффициентов и степеней
- Просмотр результатов операций

### TableWidget
Виджет для управления таблицами:
- Просмотр содержимого таблиц
- Добавление/удаление полиномов
- Переключение между типами таблиц

### ExpressionWidget
Виджет для вычисления выражений:
- Ввод постфиксных выражений
- Вычисление с использованием полиномов из таблиц
- Отображение результатов

## Сборка проекта

### Требования
- C++17 или выше
- CMake 3.16+
- Qt6 (опционально, для GUI)
- Google Test (включен как submodule)

### Команды сборки
```bash
# Клонирование с submodules
git clone --recursive <repository_url>

# Создание директории сборки
mkdir build && cd build

# Конфигурация (только библиотека)
cmake ..

# Конфигурация с Qt GUI
cmake -DBUILD_QT_GUI=ON ..

# Сборка
make -j4

# Запуск тестов
./test/test

# Запуск GUI (если собран)
./qt_gui/polynomial_qt_gui
```

## Примеры использования

### Базовые операции с полиномами
```cpp
#include "polinoms/TPolinom.hpp"

// Создание полинома P = 2*x^2 + 3*y - 1
TPolinom P({
    TMonom(2.0, {2.0, 0.0, 0.0}),  // 2*x^2
    TMonom(3.0, {0.0, 1.0, 0.0}),  // 3*y
    TMonom(-1.0, {0.0, 0.0, 0.0})  // -1
});

// Дифференцирование по x
TPolinom dPdx = P.differentiate(0);  // 4*x

// Интегрирование по y
TPolinom intPdy = P.integrate(1);    // 2*x^2*y + 1.5*y^2 - y
```

### Работа с таблицами
```cpp
#include "polinoms/TArrayTable.hpp"

TArrayTable<TPolinom> table;

// Добавление полиномов
table.insert("P1", P);
table.insert("P2", Q);

// Поиск полинома
auto result = table.find("P1");
if (result.has_value()) {
    TPolinom found = result.value();
    // Работа с найденным полиномом
}
```

### Вычисление выражений
```cpp
#include "polinoms/TPostfixCalculator.hpp"

TPostfixCalculator calc;
calc.setTable(&table);

// Вычисление P1 + P2 * 2
std::string expr = "P1 P2 2 * +";
auto result = calc.evaluate(expr);
```

## Переменные

Библиотека работает с тремя переменными:
- **x** (индекс 0) - первая переменная
- **y** (индекс 1) - вторая переменная  
- **z** (индекс 2) - третья переменная

Вывод использует символьные обозначения x, y, z вместо x0, x1, x2.
