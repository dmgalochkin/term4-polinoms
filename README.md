# Лабораторная работа: Алгебра полиномов

Система для работы с полиномами от нескольких переменных с дробными показателями степеней.

## Возможности

- Работа с полиномами от 3 переменных (x, y, z)
- Арифметические операции: сложение, вычитание, умножение
- Дифференцирование и интегрирование по любой переменной
- 6 типов таблиц для хранения полиномов
- Вычисление постфиксных выражений
- **Современный графический интерфейс на Qt6**

## Примеры использования

### Создание полиномов
```cpp
// Моном: 2.5 * x^0.5 * y^1.3
TMonom m1(2.5, {0.5, 1.3, 0.0});

// Полином из нескольких мономов
TPolinom P({m1, m2, m3});
```

### Арифметические операции
```cpp
TPolinom result = P1 + P2 * 3.0;
```

### Дифференцирование и интегрирование
```cpp
TPolinom dP = P.Differentiate(0);  // по x
TPolinom intP = P.Integrate(1);    // по y
```

### Постфиксные выражения
```cpp
// В консоли или Qt GUI
"2 * pol1 + 2 * pol2 + 3.6 * q - const6"
// Результат: 14.4x² - 6.0
```

## Типы таблиц

1. **Linear Array Table** - линейная таблица на массиве
2. **Linear List Table** - линейная таблица на списке  
3. **Ordered Array Table** - упорядоченная таблица на массиве
4. **AVL Tree Table** - таблица на AVL дереве
5. **Hash Table (Open Addressing)** - хеш-таблица с открытой адресацией
6. **Hash Table (Chaining)** - хеш-таблица с цепочками

## Архитектура

Проект построен по модульной архитектуре:
- **Ядро системы**: классы TMonom, TPolinom
- **Хранение данных**: интерфейс TTable с различными реализациями
- **Вычисления**: TPostfixCalculator для обработки выражений
- **Управление**: TPolinomManager для координации всех компонентов
- **Интерфейсы**: консольный и графический (Qt6)
